# Configure inbox

## Thunderbird

### Installation

Simply run
```bash
brew install --cask thunderbird
```

### How to add an inbox

Got to `tools` then select `account parameters`, `account managing`, `add a new inbox account`.

You must now fill in the according parameters with respect to the server's parameters.
